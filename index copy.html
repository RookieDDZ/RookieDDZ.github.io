Data Analytics for Untappd

Concept...

Since drinking/enjoying adult beverages has always been a very social event, we should be able to scrape some of the data from our favorite beer tracking app, Untappd.

How it works...

Type in the username of the person you want to review and the page will do the rest. The page will show you what styles have been enjoyed, along with a few other items.

This site is built using Jekyll, and based on the Slate theme. It is published on GitHub pages.
